Xubuntu 16.04 Packages
======

Running this script uninstalls the following packages:

- firefox
- libre office
- pidgin
- catfish
- mousepad
- notes
- onboard
- orage
- gnome-mines
- gnome-sudoku
- simple-scan
- ristretto
- evince
- transmission
- tunderbird
- screenshooter
- imagemagick
- parole
- pulse
- dictionary

And installs the following packages:

- google-chrome-stable
- sublime text 3
- numlockx
